# Graefbos Plantenkaart — prototype 1

Deze eerste versie kan:

- een satellietkaart of stratenkaart tonen;
- je huidige locatie opvragen;
- een plant toevoegen met naam, Latijnse naam, aanplantjaar, kweker, notities en identificatiestatus;
- een marker na het opslaan verslepen;
- planten lokaal in de browser bewaren.

## Belangrijke beperking

De gegevens staan voorlopig alleen in de browser waarin je ze invoert. Ze worden dus nog niet automatisch gedeeld tussen iPhone en computer. Dat wordt de volgende sprint: een gratis online database koppelen.

## Testen

Open `index.html` op een computer om de interface te bekijken. Voor betrouwbare locatiebepaling op iPhone moet het bestand via een beveiligde `https://`-website worden gepubliceerd, bijvoorbeeld met GitHub Pages.
